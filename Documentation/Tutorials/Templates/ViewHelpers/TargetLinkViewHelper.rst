Examples
^^^^^^^^^^^^^

Basic Example
""""""""""""""""""

{relatedLink.uri} is defined as "123 _blank"

Code: ::

	 <f:link.page pageUid="{relatedLink.uri}" target="{n:targetLink(link:relatedLink.uri)}">Link</Link>


Output: ::

	 A link to the page with uid 123 and target set to "_blank"
