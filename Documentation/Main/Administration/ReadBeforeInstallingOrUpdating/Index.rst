Read before installing or updating!
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Before installing this extension or updating to a new major release,
you should  **always** read the chapter “Breaking changes” at the end
of the manual and the wiki page `http://forge.typo3.org/projects
/extension-news/wiki/Release\_Notes <http://forge.typo3.org/projects
/extension-news/wiki/Release_Notes>`_ !


**Update script**
"""""""""""""""""

Sometimes it is required to use the so called “Update script”. Use
this one **right after** installing the new version.

You can find it in the Extension manager.

#. Switch to the extension manager

#. Search for the extension “news”

#. See the link “Check update script” as shown in the screenshot below
